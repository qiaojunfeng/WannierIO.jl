# Wannier90 files

```@meta
CurrentModule = WannierIO
```

!!! warning

    Several APIs (for example [`read_amn`](@ref), [`write_amn`](@ref),
    [`read_chk`](@ref), and [`write_chk`](@ref)) support Fortran unformatted
    (binary) files. Those files are compiler/platform dependent, so portability
    across all environments is not guaranteed.

    Prefer text formats for maximum interoperability, and use binary when you
    need compatibility with a specific upstream/downstream toolchain.

    Current tests cover:
    - Linux + gfortran 11.2
